# Tetris（俄罗斯方块）

一个使用 Pygame 开发的经典俄罗斯方块游戏，包含完整的现代俄罗斯方块机制（SRS、锁定延迟、DAS/ARR 等）。

## ✨ 特性

- **标准方块形状**：I、O、T、L、J、S、Z 七种形状
- **现代旋转系统**：基于 SRS（Super Rotation System），支持踢墙
- **丰富的操作**：左右移动、旋转、软降、硬降、暂停、重开
- **舒适的手感**：DAS（延迟自动移动）、ARR（自动重复速率）和锁定延迟
- **游戏节奏**：随等级提升下落速度加快，等级随消除行数增加
- **计分系统**：单行 40 分，双行 100 分，三行 300 分，四行 1200 分，乘以（等级+1）
- **视觉效果**：
  - 阴影预览（显示方块落点）
  - 下一个方块预览
  - 侧边栏显示分数、等级、消除行数
  - 控制面板提示
- **游戏状态**：游戏中、暂停、游戏结束

## 📦 依赖项

- Python 3.6 及以上
- Pygame 2.0 及以上

安装依赖：
```bash
pip install pygame
```

## 🚀 运行方法

1. 下载或克隆本仓库。
2. 确保目录下存在 **Courier.ttc** 字体文件（程序将使用该字体渲染文字）。你可以从系统字体目录中复制一个，或替换为其他等宽字体（修改代码中的 `font_path` 变量）。
3. 在终端中执行：
   ```bash
   python tetris.py
   ```

> **提示**：如果没有字体文件，程序会退出并报错。你可以将任何 `.ttc` 或 `.ttf` 文件重命名为 `Courier.ttc` 放在同一目录下，或修改代码中的字体路径。

## 🧩 游戏机制

- **方块生成**：采用“7 种形状随机包”（Bag Randomization），保证连续 7 个方块不会重复。
- **旋转系统**：使用 SRS 踢墙表，旋转时若发生碰撞会尝试偏移。
- **移动手感**：DAS 延迟（10 帧）后，ARR 速度（2 帧/格）自动重复移动。
- **锁定延迟**：方块落地后提供 30 帧的延迟时间，期间可以继续移动/旋转，延迟结束后锁定。
- **消行与升级**：消除 10 行升一级，下落速度加快（共 30 档速度）。
- **阴影预览**：显示当前方块硬降后的最终位置（半透明方块）。
- **暂停与重开**：按 `P` 暂停，按 `R` 重置游戏。

## 🛠️ 自定义与扩展

- **修改窗口尺寸**：编辑 `SCREEN_WIDTH` 和 `SCREEN_HEIGHT` 常量。
- **修改网格大小**：调整 `GRID_SIZE`、`GRID_WIDTH`、`GRID_HEIGHT`。
- **修改游戏速度**：在 `FALL_FRAMES` 数组中调整每等级下落间隔帧数。
- **修改计分规则**：编辑 `SCORES` 字典。
- **修改颜色方案**：调整 `COLORS` 和 `UI_COLORS` 中的 RGB 值。
- **修改控制按键**：在 `CONTROLS` 列表和 `handle_piece_input` 方法中修改。

## ❓ 常见问题

**Q：游戏启动时提示找不到字体文件？**  
A：请将任意一个等宽字体文件（如 `Courier New.ttf`）复制到游戏目录并命名为 `Courier.ttc`，或修改代码中的 `font_path` 变量。

**Q：为什么有时旋转会偏移？**  
A：这是 SRS 旋转系统的特性，用于模拟真实俄罗斯方块的旋转逻辑，避免穿墙。

**Q：如何调整锁定延迟时间？**  
A：修改 `lock_delay_frames` 变量（默认 30 帧，即 0.5 秒，60帧率）。
