# oh-my-tetris

一个使用 Pygame 开发的经典俄罗斯方块游戏，包含完整的现代俄罗斯方块机制（SRS、锁定延迟、DAS/ARR 等）。

## 🎯 核心玩法

- 采用 **10×20** 标准游戏区域，七种经典方块（I、O、T、L、J、S、Z）随机掉落。
- 通过移动和旋转方块，填满水平行即可消除并得分。
- 消除行数越多，等级越高，下落速度越快。
- 方块堆叠至顶部无法放置新方块时，游戏结束。

## ✨ 特性一览

- **标准方块形状**：I、O、T、L、J、S、Z 七种形状，每种拥有独立的颜色。
- **现代旋转系统**：基于 SRS（Super Rotation System），支持踢墙，旋转手感接近官方游戏。
- **舒适的操作手感**：DAS（延迟自动移动，10 帧）、ARR（自动重复速率，2 帧/格），移动平滑无延迟。
- **锁定延迟机制**：方块落地后提供 30 帧的缓冲时间，期间可继续移动或旋转，防止误触。
- **随等级提升的难度**：消除 10 行升一级，下落速度逐步加快（共 30 档速度）。
- **计分系统**：单行 40 分，双行 100 分，三行 300 分，四行 1200 分，分数乘以（等级+1）。
- **视觉效果**：
  - 半透明阴影预览，显示方块硬降后的落点。
  - 下一个方块预览区，方便提前规划。
  - 侧边栏实时显示分数、等级、消除行数。
  - 右侧控制面板展示操作说明和小提示。
- **游戏状态**：支持游戏中、暂停、游戏结束三种状态，按键切换。

## 📦 环境与依赖

- Python 3.6 或更高版本
- Pygame 2.0 或更高版本

安装依赖：
```bash
pip install pygame
```

## 🚀 快速开始

1. 克隆或下载本仓库。
2. 确保目录下存在 **Courier.ttc** 字体文件（程序将使用该字体渲染文字）。你可以从系统字体目录中复制一个等宽字体（如 `Courier New.ttf`），并将其重命名为 `Courier.ttc` 放入游戏目录。
3. 在终端中运行：
   ```bash
   python tetris.py
   ```

> **提示**：如果没有字体文件，程序会退出并报错。你也可以修改代码中的 `font_path` 变量指向其他字体文件。

## 🛠️ 自定义与扩展

- **修改窗口尺寸**：调整 `SCREEN_WIDTH` 和 `SCREEN_HEIGHT` 常量。
- **修改网格大小**：更改 `GRID_SIZE`、`GRID_WIDTH`、`GRID_HEIGHT`。
- **调整游戏速度曲线**：编辑 `FALL_FRAMES` 数组，数值越小下落越快。
- **修改计分规则**：编辑 `SCORES` 字典，自定义不同消除行数的分数。
- **更换颜色方案**：修改 `COLORS`（方块颜色）和 `UI_COLORS`（界面配色）中的 RGB 值。
- **改变按键绑定**：在 `CONTROLS` 列表和 `handle_piece_input` 方法中修改对应按键。

## ❓ 常见问题

**游戏启动时提示找不到字体文件？**  
请将任意一个等宽字体文件（如 `Courier New.ttf`）复制到游戏目录并命名为 `Courier.ttc`，或修改代码中的 `font_path` 变量。

**为什么有时旋转会偏移？**  
这是 SRS 旋转系统的特性，用于模拟真实俄罗斯方块的旋转逻辑，避免方块卡入墙中。

**如何调整锁定延迟时间？**  
修改 `lock_delay_frames` 变量（默认 30 帧，即 0.5 秒，60帧率）。

**游戏速度太快/太慢？**  
可以修改 `FALL_FRAMES` 数组中的值，数字越大下落越慢。也可以调整 `level` 对应的速度档位。
